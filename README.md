# Great Bus Express (Bus Ticket Booking System)

A C++ console-based Bus Ticket Booking System developed for the CSC415 â€“ Fundamentals of Computer Problem Solving course.  
This project simulates a real bus ticket reservation service, allowing users to select seats, choose travel time, pick destinations, and receive student discounts.

---
## Features
- Seat selection (Single or Double seats)
- Display of available seat numbers
- Date and time selection
- Location-based fare calculation
- Student discount (15%)
- Full receipt generation
- Input validation for: date, seat availability, location code

---
##  Technologies Used
- C++
- Arrays
- Loops (`for`, `do-while`)
- Conditional statements (`if`, `switch`)
- Functions & modular programming
- Input validation
